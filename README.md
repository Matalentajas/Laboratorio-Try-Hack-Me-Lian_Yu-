# Laboratorio-Try-Hack-Me-Lian_Yu-
Este repositorio contiene la documentación completa y detallada (write-up) del desafío de ciberseguridad **Lian\_Yu** de TryHackMe.  La máquina, de dificultad fácil, está fuertemente tematizada en la serie *Arrow* de DC Comics (Oliver Queen, Slade Wilson/Deathstroke), y pone a prueba habilidades fundamentales de un pentester.
